# PlanktonRun

Este é um jogo simples de corredor infinito (como o Dino-Run) desenvolvido com HTML, CSS e JavaScript puro. O jogador controla o Plankton, que deve pular sobre ondas sonoras geradas proceduralmente para sobreviver e acumular a maior pontuação possível.

Como Jogar:

1.  Pular: Use a tecla Espaço (ou clique/toque na tela) para fazer o Plankton pular.
2.  Objetivo: Desviar das ondas sonoras que vêm em sua direção pela linha central.
3.  Dificuldade Progressiva: A velocidade do jogo aumenta a cada 3 pulos, tornando-o progressivamente mais desafiador.

Para rodar o projeto localmente, basta:

1. Baixar os arquivos.
2.  Navegar até a pasta do projeto.
3.  Abrir o arquivo `index.html` em qualquer navegador.
